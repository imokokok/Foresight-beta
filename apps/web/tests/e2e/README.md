# E2E 测试用例

本文档描述 Foresight 项目的端到端测试范围和用例。

## 测试范围

### 1. 主页
- [ ] 页面加载成功
- [ ] 导航栏显示正常
- [ ] 市场列表展示正确
- [ ] 响应式布局正常

### 2. 市场页面
- [ ] 市场列表加载
- [ ] 市场详情展示
- [ ] 订单簿显示
- [ ] 价格图表展示
- [ ] 交易功能

### 3. 认证流程
- [ ] 登录页面加载
- [ ] 邮箱登录
- [ ] 钱包连接
- [ ] 会话管理
- [ ] 退出登录

### 4. 交易流程
- [ ] 下单功能
- [ ] 订单确认
- [ ] 订单取消
- [ ] 订单历史查看

### 5. 用户界面
- [ ] 钱包余额显示
- [ ] 持仓展示
- [ ] 交易历史
- [ ] 个人设置

### 6. 移动端
- [ ] 响应式布局
- [ ] 触摸交互
- [ ] 移动端导航

## 运行测试

```bash
# 运行所有 E2E 测试
npm run test:e2e

# 运行指定测试文件
npm run test:e2e -- example.spec.ts

# 生成测试报告
npm run test:e2e -- --reporter=html

# 在 CI 环境中运行
CI=true npm run test:e2e
```

## 测试数据

测试使用以下测试账户：

- 测试钱包地址：`0xTestAddress123456789012345678901234567890`
- 测试代币：使用本地 Hardhat 网络的测试代币

## 注意事项

1. 运行测试前确保本地开发服务器已启动
2. 测试网络请求使用 Mock 数据
3. 敏感操作（真实交易）不包含在 E2E 测试中
4. 每个测试用例应独立运行，不依赖其他测试状态
